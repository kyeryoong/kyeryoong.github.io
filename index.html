Working Experience

담당 업무

  • 협업 플랫폼 GAIA Works - 토크(메신저) 서비스 개발

    • GAIA Works는 토크(메신저), 메일, 캘린더, 미팅, 드라이브, 오피스, 프로젝트 앱으로 이루어진 협업 플랫폼 입니다.

    • 주요 개발 항목

      • 이미지 뷰어 개발
      • 동영상 플레이어 개발
      • 파일 업로드/다운로드 서비스 개발 및 개선
      • 다국어 처리 기능 적용
      • 타 앱 연동 기능 제공(캘린더, 미팅, 드라이브, 오피스)

    • 사용 기술

      • 프론트엔드: React + TypeScript
      • 스타일: styled components
      • 번들러: Rollup.js
      • 상태 관리: MobX

  • B2B 고객사 프로젝트 수행

    • B2C GAIA Works 제품을 기반으로 B2B 고객사의 요청에 맞춰 신규 기능과 커스터마이징된 기능을 추가적으로 제공하고 있습니다.

    • 고객사별 프로젝트

      • 서울특별시교육청 센클라우드 (2023년 1분기 ~ 진행 중)
      • 한국농어촌공사 KRC TALK (2024년 1분기 ~ 진행 중)
      • 푸디스트 성공식당 커뮤니티 (2023년 2분기 ~ 2024년 3분기)

  • 계열사 전용 토크(메신저) 컴포넌트 개발 및 유지 보수

    • 다른 계열사의 서비스에 채팅 기능을 제공하고 있습니다. 각 서비스에서 원하는 방향으로 개발할 수 있도록 헤더/바디/푸터에 필요한 기능만 표시하거나, 아이콘/배경색/글자색과 같이 스타일을 수정할 수 있습니다.

    • 주요 개발 항목

      • TmaxCloud 사 화상 미팅 - 채팅 기능
      • TmaxCloud 사 클라우드 서비스 플랫폼 - 1:1 상담원 채팅 기능
      • TmaxGAIA 사 드라이브 앱 - 채팅 기능
      • TmaxAI 사 인공지능 챗봇 - 채팅 기능
